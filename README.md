# nmap文件夹

**扫描账号脚本,扫描出的账号存在当前目录usernum文件中**

`sh check_login.sh`

**自动选随机账号登录**

`sh login.sh`

 **注销账号**

`sh logout.sh`

**放在/etc/profile.d/ 实现开机自启动login.sh脚本**

`login_start.sh`

**检测连接wifi的客户端**

`check-wifi.sh`
